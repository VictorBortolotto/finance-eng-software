# Controle Financeiro Pessoal

## Sobre o projeto

O Controle Financeiro Pessoal é uma aplicação web desenvolvida como projeto da pós-graduação, com o objetivo de auxiliar usuários no gerenciamento de suas finanças de forma simples e intuitiva.

A aplicação permite o cadastro e gerenciamento de receitas e despesas, além da visualização do saldo disponível e do histórico de movimentações financeiras, proporcionando maior controle sobre as finanças pessoais.

O projeto foi concebido utilizando conceitos de **Lean Inception**, definindo um **MVP (Minimum Viable Product)** focado nas funcionalidades essenciais para validar a proposta da solução.

## Funcionalidades

- Cadastro e autenticação de usuários;
- Cadastro, edição e exclusão de receitas;
- Cadastro, edição e exclusão de despesas;
- Visualização das movimentações financeiras;
- Consulta de saldo atualizado;
- Resumo financeiro;
- Filtro de movimentações por período.

## Objetivo

O principal objetivo deste projeto é aplicar conceitos de desenvolvimento de software, engenharia de requisitos, metodologias ágeis e arquitetura de aplicações web por meio da construção de um sistema funcional para controle financeiro pessoal.
